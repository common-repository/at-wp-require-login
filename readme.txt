DSVGO/GDPR

== Description ==
Very lightweight plugin which if activated, it locks down the website for public access. Visitors will be prompted with the wp-login.php page where they should log in to access the site. Useful for under construction websites.

== Installation ==
Upload and activate plugin to your WP installation. Deactivate it if you wish to allow public access to your site.
